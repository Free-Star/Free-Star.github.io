JS入门

发表于2024-02-21|前端

基本定义

  • Java与JS的区别？
    • Java是一门面向对象的语言，而JavaScript更像是一门函数式编程语言。
  • 静态页面和动态页面的区别？
    • 静态页面和动态页面的区别在于是否与服务器进行数据交互。简单地说，页面是否用到了后端技术（如PHP、JSP、ASP.NET）。
  • 语法糖
    • 语法糖（Syntactic sugar）是计算机科学中的一个概念，指的是一种编程语言的语法特性，它并不提供新的功能，而是为了使代码更加易读、简洁或者更符合人类的直觉。

JS引入方式

  • 外部JS
  • 内部JS
    • 写在head的script标签里面
  • 元素属性JS
    • 在元素的事件属性中直接编写JS或调用函数

语法

JS语法

  • 变量与常量
    • 常量
    • 变量
      • 变量由字母、下划线、$或数字组成，并且第一个字母不能是数字。
      • 变量不能是系统关键字和保留字。
    • 语法
    • 所有JavaScript变量都由var声明。
  • 数据类型
    • 基本数据类型
      • 只能有一个值
      • 数字
      • 字符串（用单引号或双引号括起来的字符集）
        • **在用单引号括起来的字符串中，不能含有单引号，只能含有双引号。反之同理
      • 布尔值
      • 未定义值
        • var声明但没有赋值的变量，值都是undefined。
      • 空值
        • null,系统没有给其分配空间
    • 引用数据类型
      • 可以有多个值
      • 对象（数组也是属于对象的一种）
  • 运算符
    • 算术运算符
      • 加法运算符
        • 算数
        • 字符串拼接
        • 这个算法遵循数据类型优先级，比如说字符串和数据类型运算后得到的结果是字符串类型
      • 自增运算符
        • ++在左，先增后算，++在右，先算后增。
      • 自减运算符（同理）
    • 赋值运算符
      • =
      • +=
      • -=
        • =
      • /=
      • 不多做解释：a+=b等同于a=a+b
    • 比较运算符
      • <

      • =

      • <=
      • ==
      • !=
    • 逻辑运算符
      • 与运算
        • “&&”
        • 全真则真，有假则假
      • 或运算
        • “||”
        • 全假则假，有真则真
    • 条件运算符
      • var a=条件?表达式1 :表达式2
      • 若条件为真，则a取表达式1，若为假，则a取表达式2的值。
  • 表达式与语句
    • 以分号”;”结尾的是语句，没有以分号”;”结尾的是表达式
  • 类型转换
    • 显式类型转换
      • 也称强制类型转换，需要人为进行类型转换
    • 隐式类型转换
      • JS自动进行的类型转换
    • 字符串转换为数字
      • Number( )
        • 不能接受非全数字的字符串，必须全是数字
      • parseInt( ) ; parseFloat( );
        • parseInt（　）和会对字符串从左到右进行判断。如果第1个字符是数字，则继续判断，直到出现非数字为止（小数点也是非数字）；如果第1个字符是非数字，则直接返回NaN。parseInt（　）可以接受第1个字符是加号或减号。
    • 数字转换为字符串
      • 与空字符串相加（比较常用！）
      • toString( )
  • 转义字符
    • \‘
    • \“
    • \n
      • 如果是在document.write（　）中换行，则应该用
        。如果是在alert（　）中换行，则应该用\n。
    • 以上是常用的三种转义字符

    • 单行注释
    • 多行注释

函数

  • 函数的定义
    • 没有返回值的函数
    • 有返回值的函数

      • 全局变量：全局变量一般在主程序中定义，其有效范围是从定义开始，一直到整个程序结束。
      • 局部变量：局部变量一般在函数中定义，其有效范围只限于在函数中。
  • 函数的调用
    • 直接调用
    • 在表达式中调用
    • 在超链接中调用
      • 在a元素的href属性中用“javascript：函数名”的形式来调用函数。当用户点击超链接时，就会调用该函数
    • 在事件中调用
      • 当一个事件产生的时候，我们就可以调用某个函数来针对这个事件作出响应。
  • 嵌套函数
    • 在一个函数的内部定义另外一个函数。但是在内部定义的函数只能在内部调用，如果在外部调用，就会出错。

DOM基础

  • DOM对象
    • Document Object Model（文档对象模型），是由W3C定义的一个标准。
    • DOM操作，可以简单地理解成“元素操作”。
  • DOM结构
    • DOM树
    • 我们在操作元素时，其实就是把这个元素看成一个对象，然后使用这个对象的属性和方法来进行相关操作。
  • 节点类型
    • 元素节点
    • 属性节点
    • 文本节点
    • 需要注意的点
      • 节点和元素是不一样的概念，节点是包括元素的。
      • 一个元素就是一个节点，这个节点称为“元素节点”。
      • 属性节点和文本节点看起来像是元素节点的一部分。实际上，它们是独立的节点，并不属于元素节点。
      • 只有元素节点才可以拥有子节点，属性节点和文本节点都无法拥有子节点。
  • 获取元素
    • getElementById( )
    • getElementsByTagName( )
      • oLi.length表示获取“类数组”oLi的长度，有多少个元素，长度就是多少。
    • getElementsByClassName( )
    • querySelector( )和querySelectorAll( )
    • getElementsByName( )
    • document.title和document.body
